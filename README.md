# BaseProjectForH5

### 简介

 主要依赖gulpfile, 复制src文件夹到build以及dist 实现开发环境的搭建


### 安装NodeJs

下载地址：[https://nodejs.org/en/download/](https://nodejs.org/en/download/)
一路默认安装，node自己带了npm包管理工具
建议使用4.x-5.x版本
  
  
### 安装gulp（自动化构建，依赖node）
npm install -g gulp


### 安装npm依赖包
在当前目录命令行 npm install

### 运行
gulp
